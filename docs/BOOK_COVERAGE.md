# 经典书籍主题覆盖矩阵

本项目以两本经典教材的主题结构检查覆盖面：Stevens/Rago 的 [《UNIX 环境高级编程》（Advanced Programming in the UNIX Environment）第 3 版](https://www.informit.com/store/advanced-programming-in-the-unix-environment-9780321638007)（APUE），以及 Kerrisk 的 [《Linux/UNIX 系统编程手册》（The Linux Programming Interface）详细目录](https://man7.org/tlpi/toc-detailed.html)（TLPI）。实验代码根据 API 文档独立编写，没有复制书中示例。

矩阵中的章节号帮助学习者回到书中阅读完整语义；实验只负责把一个关键行为缩小成可运行的观察点，不能代替章节内容。

## 有意保留的边界

以下主题暂不放入默认 CTest，因为它们需要额外权限、宿主配置或会产生系统级副作用：

- 用户/组凭据切换、set-user-ID 和 Linux capabilities；
- daemon、`syslog` 与服务管理器集成；
- 实时调度策略、锁页和高精度延迟基准；
- namespace、cgroup、seccomp、fanotify 与挂载操作；
- 原始/数据链路套接字和外部网络服务；
- ACL、NUMA、native Linux AIO 与 `io_uring` 等需要额外开发库的接口。

这些边界会在后续专门的“特权实验”或“可选依赖实验”中处理，不让基础学习路径依赖 root capabilities 或宿主服务。
